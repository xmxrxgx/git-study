## github에 코딩파일 업로드

1. 내 프로필 Repositories 클릭
2. New 클릭
3. Repository name에는 코딩파일 제목 작성
4. Create Repository 클릭
5. VS Code 좌측 패널에서 git 아이콘 클릭
6. Initialize Repository 클릭
7. 업로드할 파일함 + 눌러서 Staged Changes로 이동
8. Message 입력칸에 어떤 것이 변경되었는지 입력하고 commit 버튼 클릭
9. 터미널에 아래 코드 순처적으로 붙여넣기
git remote add origin ----------------------
git push -u origin main

## github에 코딩파일 수정 후 업로드

1. 업로드할 파일만 + 눌러서 staged Changes로 이동
2. Message 입력칸에 어떤 것이 변경되어쓴지 입력하고 Commit 버튼 클릭
3. [Sync Changes] 버튼 클릭하면 내 컴퓨터에서 github로 수정된 코드 전송
